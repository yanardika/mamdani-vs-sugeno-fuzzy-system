# Fuzzy Logic System: Mamdani & Sugeno
An implementation of Fuzzy Inference Systems (FIS) using both Mamdani and Sugeno methods to model rule-based decision-making systems.

This project demonstrates core concepts of fuzzy logic, membership functions, rule evaluation, and defuzzification techniques.

# Project Objective
To design and implement fuzzy inference systems using:
- Mamdani method
- Sugeno method
And analyze their differences in:
- Rule evaluation
- Output generation
- Computational approach
- Decision interpretation

# Concepts Applied
- Fuzzy Sets
- Membership Functions
- Fuzzification
- Rule Base Construction
- Inference Engine
- Defuzzification (Mamdani)
- Weighted Output (Sugeno)

# Methodology
## Mamdani FIS
- Rule-based inference
- Aggregation of fuzzy outputs
- Defuzzification (e.g., Centroid method)
## Sugeno FIS
- Rule-based inference
- Crisp output functions
- Weighted average computation
